# Code of Conduct

## Our Commitment

This repository is a professional learning and practice space focused on automation testing using Selenium. A respectful, inclusive, and constructive environment is expected for all contributors and visitors.

## Expected Behaviour

Participants are expected to:

- Be respectful and considerate in communication
- Provide constructive feedback
- Focus discussions on technical improvement and learning
- Maintain professionalism at all times

## Unacceptable Behaviour

The following will not be tolerated:

- Harassment, discrimination, or offensive language
- Personal attacks or hostile interactions
- Trolling or disruptive behaviour
- Sharing sensitive information without permission

## Scope

This Code of Conduct applies to all interactions within this repository, including issues, pull requests, and discussions.

## Enforcement

Project maintainers have the right to remove, edit, or reject contributions that do not align with this Code of Conduct.
